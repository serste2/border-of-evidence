# Visual Style

The visual language must follow Serena Stelitano's painterly and critical style rather than a generic dashboard aesthetic.

## Style anchors

- COSMOGONIA
- Andrea Grows
- MAP-style split landscape drawing supplied during project development
- Oracle as a programmable-art precedent

## Formal qualities

- painterly, irregular surfaces
- dark expressive outlines
- stratified ground and visible soil layers
- dense vegetation and ecological complexity
- acidic yet earthy palette: greens, browns, purples, ochres, magentas
- contrast between compressed extractive mass and living regenerative expansion
- hand-made atmosphere, not polished corporate flat design

## Interface rule

The UI must not overpower the artwork. The visual world is the primary interface. Panels, cards, drawers, forum modules, quiz prompts, and archive elements should behave like annotations or instruments attached to the world.

## Scene principles

Left side:
- compressed
- saturated
- industrial
- extractive
- claustrophobic
- not a future to explore

Right side:
- explorable
- layered
- regenerative
- technologically embedded but ecological
- contains solar panels integrated into structures and landscape
- contains solar drones monitoring fire, air, and soil
- can extend laterally into a living museum

## Asset strategy

Final assets should be drawn by Serena or derived from her drawing system. Placeholder assets are temporary only. Use transparent PNG or SVG layers grouped by:

- background
- middle ground
- foreground
- hotspots
- animated overlays
- UI annotation elements
