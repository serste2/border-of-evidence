# Frontend

The frontend will render the live visual world and user-facing modules.

## Planned modules

- split evidence landscape
- regenerative lateral museum
- scene-state renderer
- evidence drawers and cards
- live feed ticker
- forum interface
- archive browser
- quiz interface
- poll interface
- community gallery
- rewards / downloadable fact cards

## Visual principle

The illustrated world is the primary interface. UI overlays should annotate the world without replacing it.

## Suggested stack

Initial prototype:

- Vite
- React
- TypeScript
- CSS modules or Tailwind
- PixiJS or Three.js for the 2.5D / parallax scene when ready

Start static first, then connect to backend APIs.
